## 簡介
- 一個具有獨特功能的中原租屋網，並且也製作了一般功能。
- 特色：
1. 目的地多選功能 (夥伴開發)
2. 合租功能
- 其他功能：
1. 註冊/登入出功能
2. 查詢及篩選房屋資料功能(租屋)
3. 分析並查詢租金功能(分析)
4. 出租房屋功能(出租)
5. 簡單合租功能

### 技術
- 前端使用Django為Web框架，並以HTML、CSS、JavaScript做前端頁面設計
- 後端資料庫使用PostgreSQL
- code存於 ->Project_ChungYuan_website 分支  *連接資料庫設定已更動
